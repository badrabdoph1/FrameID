## Ownership Rule

Each feature detail must have one primary file in this directory. Core documents may link to it and describe only cross-system architecture, governance, or shared data flow.

Update an existing feature file instead of creating another document for the same system. Create a new feature document only when the capability is a distinct, material subsystem with its own components, flow, and maintenance concerns.
