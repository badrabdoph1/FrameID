# AGENTS.md

## اللغة
- يجب استخدام اللغة العربية في جميع الردود والتقارير والملخصات والتعليقات في الكود
- جميع المخرجات النصية يجب أن تكون بالعربية

## النشر التلقائي
- بعد إنهاء أي مهمة أو طلب، يجب رفع التعديلات تلقائياً (commit + push)
- رسالة الـ commit يجب أن تكون بالعربية وتعكس عنوان/موضوع الطلب
- يجب الالتزام فقط بالتعديلات الخاصة بالمهمة الحالية - لا يتم commit لأي تعديلات تخص مواضيع أو طلبات أخرى حتى لو كانت موجودة في `git status`
- في حال وجود أعدادات واقفة للنشر (مثل pre-commit hooks أو ci checks)، يتم تخطيها والتركيز على رفع التعديلات الخاصة بالمهمة الحالية فقط
- هذه القاعدة تنطبق على جميع التعديلات في أي ملف (كود، إعدادات، توثيق، إلخ)
- هذه القاعدة تطبق على جميع الجلسات المستقبلية فقط
- بعد الـ push إلى GitHub، يجب مزامنة التعديلات مع GitHub Pages من خلال GitHub Desktop (سطح المكتب)
